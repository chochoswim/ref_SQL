DBeaver shortcut
- ctrl + enter           : run
- ctrl + \               : result+
- alt + X                : run the script from top to bottom
- ctrl + /	             : comment
- alt + shift + a        : mouse shape +
- ctrl + d               : delete the line
- ctrl + shift + l       : lower case
- ctrl + shift + u	     : upper case
- 특정 라인 선택 + tab  : 한 번에 들여쓰기
